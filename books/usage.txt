usage: 
    python3 books.py [--book-search S]
    python3 books.py [--author-search S]
    python3 books.py [--publish year year]
    python3 books.py [--help]

Process books.csv using a keyword from the title, an author, or a range of publication:

--book-search, -bs S             print a list of books whose titles contain the string S
--author-search, -as S           print a list of authors whose names contain the string S
--publish, -p year year          print a list of books published between the two years
--help, -h                       show this help message and exit
