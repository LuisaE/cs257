usage: 
    python3 books.py [--book-search S]
    python3 books.py [--author-search S]
    python3 books.py [--publish year year]
    python3 books.py [--help]

These are commands to filter books.csv, a file full of data about books and authors:

--book-search S             Print a list of books whose titles contain the string S
--author-search S           Print a list of authors whose names contain the string S
--publish year year         Print a list of books published between the two years
--help                      Print this usage message
